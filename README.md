# ☕ Cafe Finder Web App

A full-stack web application that displays cafes with WiFi availability and great coffee, built using data from a REST API project.

This project uses an SQLite database (`cafes.db`) to serve and manage cafe information through a user-friendly website.

---

## 🚀 Features

* View a list of cafes with details like:

  * Location
  * Coffee rating
  * WiFi availability
  * Power sockets

* Add new cafes to the database

* Delete existing cafes

* Clean and responsive UI

* Backend powered by REST API

---

## 🛠️ Tech Stack

* Frontend: HTML, CSS, Bootstrap
* Backend: Python, Flask
* Database: SQLite (`cafes.db`)
* API: RESTful API
